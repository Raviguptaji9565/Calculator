# Modern Android-Style Calculator 📱

A clean, user-friendly, and modern calculator application designed with an Android-like interface. 

## ✨ Features
* **Basic Arithmetic:** Perform addition, subtraction, multiplication, and division effortlessly.
* **Theme Toggle:** Switch between sleek Dark mode and bright Light mode.
* **Calculation History:** Keeps a record of your past calculations for quick reference.
* **Responsive Design:** Mobile-friendly UI that looks great on any screen size.

## 🛠️ Built With
* HTML
* CSS
* JavaScript
